## Title

IPTS Mode C Diverted Site Example Reference

## Purpose

Record approved reference examples for IPTS Mode C Diverted Site formatting.

## Knowledge Statement

Pilot 002 provides Mode C examples showing:

- Google Keyword source example: `2,100,https://example.com,Claude AI`
- Customized source URL example: `0,100,https://example.com,https://www.google.com/search?q=Claude+AI`
- Wrong example: `0,100,https://www.google.com/search?q=Claude+AI`
