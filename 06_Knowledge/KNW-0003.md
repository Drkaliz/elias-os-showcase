## Knowledge ID

## Title

IPTS Mode A URL_CODE Example Reference

## Purpose

Record approved reference examples for IPTS Mode A URL_CODE formatting.

## Knowledge Statement

Pilot 002 provides Mode A examples showing:

- Standard URL_CODE example: `0,35,https://www.google.com/search?q=Claude+AI&gl=us`
- Deep Search example only when requested: `0,20,https://www.google.com/search?q=Claude+AI&start=10`
- Normal-mode wrong example: `0,25,https://www.google.com/search?q=Claude+AI&start=10`
