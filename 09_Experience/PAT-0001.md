# PAT-0001

## Pattern ID

PAT-0001

## Purpose

Provide a reusable reference pattern for organizing IPTS output decisions around mode taxonomy, source type mappings, and reviewed examples.

## Pattern Statement

When reviewing or preparing IPTS-specific assets, separate the work into:

1. Identify the IPTS output mode taxonomy.
2. Use the relevant source type mapping reference.
3. Compare candidate output structure against reviewed examples for the selected mode.
4. Treat full setup block labels as reference structure, not as automatic output instructions.

This pattern supports review organization only. It does not authorize output generation or create operational rules.
