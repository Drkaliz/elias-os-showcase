# 00_Architecture

## Purpose

Store approved architecture, constitutional authority, and architecture decision records for Elias OS.

## Responsibilities

- Preserve system design authority.
- Record approved architecture changes.
- Define governance before implementation.

## Allowed contents

- Architecture documents.
- Constitution documents.
- ADR records.
- Architecture review material.

## Forbidden contents

- Implementation code.
- Generated exports.
- Project-specific operational files.
- Unreviewed rules or knowledge entries.

## Future role inside Elias OS

Remain the governing architecture source for every future milestone.
